# ClipNote AI 🚀

ClipNote is a premium YouTube video summarization tool that uses AI to transform hours of video content into concise, readable summaries and key insights.

![ClipNote Logo](src/assets/logoicon.png)

## ✨ Features

- **AI Summarization**: Instantly generate high-quality summaries of any YouTube video.
- **Key Insights**: Extraction of critical timestamps and takeaways.
- **Smart Dashboard**: Manage your summary history and organized collections.
- **Premium UI**: Sleek dark-mode interface with glassmorphism design.
- **Export Options**: Export summaries to PDF, Markdown, or shareable links.
- **Responsive Design**: Works perfectly on Desktop, Tablet, and Mobile.

## 🛠️ Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS 4.0
- **Icons**: Lucide React
- **Animations**: Motion (Framer Motion)
- **Deployment**: Vercel & GitHub Pages

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/app-clipnote/clipnote.git
   cd clipnote
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Deployment

### Vercel (Recommended)
The project is configured for seamless deployment on Vercel. Simply push to your `main` branch, and the `vercel.json` will handle the routing and build settings automatically.

### GitHub Pages
To deploy to GitHub Pages, run the following command:
```bash
npm run deploy
```
*Note: This command uses `cross-env` to set the correct base path for the subdirectory deployment.*
